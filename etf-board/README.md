# ETF Board

배포본 화면을 기준으로 새로 구성한 정적 웹앱입니다. 원본 `Release` 폴더는 보존했고, 브라우저에서 `index.html`을 열어 바로 확인할 수 있습니다.

## 반영한 개선 방향

- 축구 포지션별 자산 성향을 제한했습니다.
- 4-3-3, 3-5-2, 4-4-2, 4-1-2-3 프리셋 포메이션을 제공합니다.
- 사용자가 공격, 미드, 수비 수를 직접 조정하는 커스텀 포메이션을 제공합니다.
- 포메이션은 총 10개 종목으로만 구성됩니다.
- 공격수는 최소 1명, 최대 4명까지 설정할 수 있습니다.
- 미드필더와 수비수는 각각 최소 1명, 최대 5명까지 설정할 수 있습니다.
- 수비는 예금, 채권, 현금성 자산만 배치할 수 있습니다.
- 미드필더는 ETF와 섹터 바스켓만 배치할 수 있습니다.
- 공격은 상위 10 주도주만 배치할 수 있습니다.
- 종목을 선택하면 3일 전부터 현재까지의 흐름과 등락을 보여줍니다.
- ETF를 선택하면 구성 종목별 상승/하락 원인을 함께 보여줍니다.
- 포메이션 전체를 기준으로 수비, 미드, 공격, ETF 구성 원인을 묶은 보고서를 생성합니다.

## 실행

`index.html`을 브라우저에서 열면 됩니다.

또는 아래처럼 로컬 서버로 실행할 수 있습니다.

```powershell
python -m http.server 5173
```

그 다음 `http://localhost:5173`으로 접속합니다.

## 데이터

현재 데이터는 학습 화면 구성을 위한 샘플 데이터입니다. 실시간 데이터 연동 단계에서는 가격, ETF 구성 종목, 뉴스, 원인 분석 데이터를 별도 API 또는 DB로 분리하면 됩니다.
